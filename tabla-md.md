| Formato        | Código Markdown                      | Ejemplo renderizado           |
|---------------|--------------------------------------|--------------------------------|
| Encabezado 1  | `# Título 1`                        | # Título 1                    |
| Encabezado 2  | `## Título 2`                       | ## Título 2                   |
| Encabezado 3  | `### Título 3`                      | ### Título 3                  |
| Negrita       | `**texto en negrita**`              | **texto en negrita**          |
| Cursiva       | `*texto en cursiva*`                | *texto en cursiva*            |
| Negrita+Cursiva| `***texto combinado***`            | ***texto combinado***          |
| Lista viñetas | `- Elemento`                        | - Elemento                    |
| Lista numerada| `1. Elemento`                       | 1. Elemento                   |
| Código en línea| `` `código` ``                     | `código`                      |

| Enlace        | `https://ejemplo.com`      | https://ejemplo.com  |
| Imagen        | `https://ejemplo.com/img.png`| https://ejemplo.com/img.png |
| Cita          | `> texto citado`                    | > texto citado                |
